# Portfolio Website

A modern, responsive personal portfolio website built with React, Tailwind CSS, and Framer Motion. This project showcases a developer's skills, projects, and experience with a sleek, dark-themed glassmorphism design.

![Portfolio Screenshot](https://via.placeholder.com/800x400?text=Portfolio+Preview)

## üöÄ Features

-   **Hero Section**: Interactive terminal animation and "Available for work" status.
-   **Tech Stack Grid**: A clean visual representation of skills and technologies.
-   **Stats Cards**: Highlights experience and LeetCode achievements.
-   **Projects Showcase**: Featured projects with GitHub and Live Demo links.
-   **Contact CTA**: A prominent call-to-action card for collaboration.
-   **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
-   **Animations**: Smooth transitions and effects using Framer Motion.

## üì¶ Installation & Run

1.  **Clone the repository** (or download source):
    ```bash
    git clone https://github.com/yourusername/portfolio-app.git
    cd portfolio-app
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Build for production**:
    ```bash
    npm run build
    ```

## üé® Customization

You can easily customize the content by editing the components in `src/components/`.
-   **Personal Info**: Update `Hero.tsx` and `Contact.tsx`.
-   **Skills**: Update the `skills` array in `About.tsx`.
-   **Projects**: Update the `projects` data in `Projects.tsx`.
